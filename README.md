# D4NH's dotfiles

* OS: `macOS Sierra 10.12.4`
* Window Manager: `kwm`
* Terminal: `iTerm`
* Terminal Font: `Source Code Pro for Powerline`
* Terminal color scheme: `base16-shell`
* Text Editor: `Sublime Text 3 + VIM`
* Editor Font: `Source Code Pro for Powerline`
* File Browser: `Ranger`
* Web Browser: `Google Chrome`
